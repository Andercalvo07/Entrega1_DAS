🏋️ TrainUp – Tu Compañero de Entrenamiento

MiGym es una aplicación Android diseñada para ayudarte a planificar, organizar y maximizar tus sesiones de ejercicio de manera sencilla y eficiente.

✨ Características Destacadas

📋 Gestión completa de entrenamientos: Lleva un registro detallado de tus rutinas.

🔔 Recordatorios inteligentes: Sistema de notificaciones para que nunca olvides tu entrenamiento.

🌍 Multiidioma: Disponible en español e inglés.

🎨 Interfaz moderna: Diseño elegante basado en Material Design.

💾 Base de datos local: Gestión eficiente de tus datos con Room.

📱 Responsivo y adaptable: Funciona perfectamente en cualquier tamaño de pantalla.

🛠 Requisitos Técnicos

Antes de instalar MiGym, asegúrate de contar con:

Android Studio: Arctic Fox o superior

Gradle: 7.0 o superior

Android SDK: 21 o superior

🚀 Instalación Rápida

Clona el repositorio:

git clone https://github.com/andercalvo07/trainup.git

Abre el proyecto en Android Studio.

Sincroniza Gradle para descargar dependencias.

Ejecuta la app en un emulador o dispositivo físico.

📂 Estructura del Proyecto
app/src/main/java/com/example/migym/
├─ data/          → Entidades y DAOs
├─ dialogs/       → Diálogos de la app
├─ repositories/  → Repositorios de datos
├─ ui/            → Actividades y fragmentos
└─ utils/         → Utilidades y helpers
